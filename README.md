# Final Project - Recipe App

## Setup (Run these in seperate terminals!)

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev

## Features
- User registration & login
- Password change
- Recipe CRUD
- Dashboard UI

## Testing
1. Register user
2. Login
3. Add recipe
4. Delete recipe
5. Change password
6. Logout and login again
